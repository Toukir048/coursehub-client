# CourseHub Client

CourseHub is a responsive course discovery and management platform built with React and TypeScript.

## Features

- Responsive landing page
- Course search, filtering, sorting, and pagination
- Public course details page
- Login and registration interfaces
- Demo login
- Protected add course page
- Protected manage courses page
- Dashboard statistics and charts
- About, Contact, Blogs, Privacy, and Terms pages
- Responsive navbar and footer

## Technology

- React
- TypeScript
- Vite
- Tailwind CSS
- daisyUI
- React Router
- Recharts
